# Anand Samuel Paul - Portfolio

A modern, responsive portfolio website showcasing cybersecurity expertise, projects, and professional experience.

## 🚀 Quick Deploy to Vercel

1. **Push this entire folder to GitHub**
2. **Go to [vercel.com](https://vercel.com)** and sign in with GitHub
3. **Click "New Project"**
4. **Import your repository**
5. Vercel will auto-detect React and deploy automatically!

## 🛠️ Local Development

```bash
# Install dependencies
npm install

# Run locally
npm start

# Build for production
npm run build
```

## 🎨 Features

- Modern dark theme with cyberpunk aesthetic
- Smooth scroll navigation
- Responsive design for all devices
- Interactive hover effects
- Sections: Hero, About, Experience, Projects, Skills, Education, Certifications

## 🔧 Customization

Edit `src/Portfolio.jsx` to customize:
- Colors (CSS variables at top)
- Content (projects, experience, skills)
- Styling (all inline in the component)
